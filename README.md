# Minesweeper
A Minesweeper game implemented in Java with a graphical user interface.
This project is a classic Minesweeper game implemented in Java with a graphical user interface. The goal of the project was to recreate the well-known gameplay while applying object-oriented programming principles and designing a clean game architecture.

## Features

- Randomly generated game board
- Classic Minesweeper gameplay mechanics
- Cell revealing with recursive opening of empty areas
- Flagging system for suspected mines
- Game state management (win / loss detection)
- Graphical user interface for interacting with the board

## Technologies

- Java
- Object-Oriented Programming (OOP)
- GUI framework (Swing)

## Purpose

The project was created as a programming exercise to practice object-oriented design, game logic implementation, and GUI development in Java.

## How to Run

1. Clone the repository
2. Open the project in your preferred Java IDE
3. Run the main application class
